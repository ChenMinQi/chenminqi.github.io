Koch曲线的动态化

By Apple

2015-07-06 更新日期:2015-07-06

文章目录

上周在Matrix67的博客里看到了一个有点意思的博文，《7 个分形图形的动画演示》，而Matrix67并没有给出他的源代码，外加我曾经写过Koch曲线的代码，只差一个动态化的过程，所以就决定补一个代码。

初稿，无说明，只有源码

Clear["Global`*"]
rotate[{x1_, y1_}, {x2_, y2_}] := Evaluate[Simplify@RotationTransform[1. Pi/3, {x2, y2}][{x1, y1}]];
generate[{p1_, p5_}] := Module[{p2, p3, p4},
    p2 = (p5 - p1)/3 + p1;
    p4 = 2 (p5 - p1)/3 + p1;
    p3 = rotate[p4, p2];
    {p1, p2, p3, p4}]
data[0] = N@{ {0, 0}, {1, 0} };
data[n_] := data[n] = Flatten[{generate /@ Partition[data[n - 1], 2, 1], { { {1, 0} } }}, 2];
get[{p1_, p2_, p3_, p4_, p5_}, t_] := { {p1, p2, (1 - t) p4 + t p3}, {(1 - t) p2 + t p3, p4, p5}}
getGIF[data_, t_] := get[#, t] & /@ Partition[data, 5, 4];
newdata[t_] := Flatten[getGIF[data[Quotient[t + 1, 1]], Mod[t, 1]], 1];

Manipulate[ListLinePlot[newdata[t], PlotRange -> { {0, 1}, {-0.02, 0.3}}, 
  AspectRatio -> 0.32, Axes -> False, PlotStyle -> RGBColor[0.353, 0.741, 0.913], 
  ImageSize -> {500, 200}], {t, 0, 4, 0.03}, SaveDefinitions -> True, 
 AutorunSequencing -> All]

alldata = Table[newdata[t], {t, 0, 4, 0.02}];
pic = ListLinePlot[#, PlotRange -> { {0, 1}, {-0.02, 0.3}}, AspectRatio -> 0.32, Axes -> False, 
     PlotStyle -> RGBColor[0.353, 0.741, 0.913], ImageSize -> {500, 200}] & /@ alldata;
Export[NotebookDirectory[] <> "Koch.gif", pic, "DisplayDurations" -> 1/60]
